# RAG-RAIF
RAG-агент для финала хакатона "AI for Finance"

# О проекте

Бобик - агент по финансовой грамотности для детей и подростков. Он был создан, чтобы помочь молодежи разобраться в тонкостях финансовой системы, удовлетворить и повысить спрос  к знаниям по финансовой грамотности, а также защитить от нападок мошенников и недоброжелателей в интернете.

Наша основная цель заключается в том, чтобы  научить молодое поколение осознанно пользоваться современными  финансовыми инструментами, понимать закономерности и взаимосвязи. А также подарить спокойствие их родителям, которые будут точно знать - их ребенок в хороших руках!

# Как запустить агента?

 1)  скачайте весь репозиторий.
 2) откройте ``` main.py ``` и  ``` syte.py ```
 3) в командной строке пропишите ``` streamlit run syte.py ``` - готово! нужно немного подождать, и агент прогрузится.

# Устройство модели:

- Агент построен на:

- Langchain - фреймворк для работы с LLM-моделями.
- LangGraph для описания текущих состояний модели - фреймворк для оркестровки агентных систем.
- RERANKER cross-encoder/ms-marco-MiniLM-L-6-v2
  
 **агентная часть**
- MAIN: gemini-2.0-flash-001 (вызывается с помощью API-ключа)
- JUDGE: mistral-small-3.2-24b-instruct (вызывается с помощью API-ключа)
- EMBEDDINGS: text-embedding-3-small  (вызывается с помощью API-ключа)


  Итого RAG обладает несколькими свойствами:
- Проверка на галлюцинации
- Маршрутизация
- Цензура
- Проверка ответа на вопрос
- Обработка тегов
